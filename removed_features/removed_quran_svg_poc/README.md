Archived Quran SVG POC tools
============================

The current production-like generator remains active at:

- `tools/qpc_svg_poc/generate_quran_only_svg_sample.py`

The scripts in `tools/` here are older POC helpers for the pre-Quran-only
pipeline, one-page probes, and WebP compression experiments. They were moved
out of the active tools folder so future generation uses the corrected
Quran-only pipeline that removes only external non-Quranic page chrome while
keeping internal surah titles, basmallah, Quran words, and ayah marks.
